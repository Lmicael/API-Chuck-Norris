Documentação API Chuck Norris Jokes

GET /api/logs

Recupera logs com base em parâmetros opcionais:

  • startDate (Opcional): Filtra os registros desta data (YYYY-MM-DD).
  • endDate (Opcional): Filtra os registros até esta data (YYYY-MM-DD).
  • Se nenhum parâmetro for fornecido, retorna a lista completa de logs.

Listar todos os registros de logs:

GET "http://localhost:3000/api/logs"

Exemplo de Resposta:

[
    {

GET /api/jokes/random

Recupera uma piada aleatória de Chuck Norris.

Obtenha uma piada aleatória:

GET "http://localhost:3000/api/jokes/random"

Exemplo de Resposta:

{
  "joke": "Chuck Norris doesn't read books. He stares them down until he gets the information he wants."
}

GET /api/jokes/search

Recupera piadas de Chuck Norris contendo a string de consulta especificada.

Parâmetros de consulta:

  • query: Termo para pesquisar em piadas.

Procure piadas que contenham "money":

GET "http://localhost:3000/api/jokes/search?query=money"

Exemplo de Resposta:

{
    "joke": "When Chuck Norris calls 1-900 numbers, he doesn't get charged. He holds up the phone and 
    money falls out."
}
